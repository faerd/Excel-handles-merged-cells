# EXCEL for mac 的宏涉及的功能

1.MergeKeepFirst
合并单元格并保留首个内容

2.BatchMergeDown
向下批量合并单元格（智能合并）

3.MergeKeepAll
合并选中单元格并保留所有内容（分段）

4.BatchMergeBlanks
向下批量合并空白单元格

5.MergeAndFill
合并单元格并填充（先行后列）
没问题。

6.SplitCellContent
拆分单元格内容 

7.UnMergeAndFill
取消合并并自动填充

8.FillDownBlanks
自动向下填充

9.MergeCellContents
内容合并到首个单元格，其他清空，不改变合并状态

10.ddFixedRowHeight
弹出对话框，给选中行增加固定行高

11.SmartSetRowHeight
智能行高：最小行高+自动调整+固定增加值，支持合并单元格

12.SetStandardFormat
标准页面设置：A4，指定边距，页脚页码格式

13.SetPrintTitleRows
设置选中行为顶端标题行（处理合并单元格）

14.ClearPrintTitleRows
取消顶端标题行设置

15.PopUndo
撤销管理模块（支持3级撤销） 

16.创建备份
